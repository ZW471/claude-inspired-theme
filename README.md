# Claude Inspired Theme

A warm, elegant VS Code theme inspired by [Claude AI's](https://claude.ai) signature cream and terracotta color palette. Includes both **light** and **dark** variants.

## Themes

### Claude Light

A soft, cream-toned light theme with warm accents.

![Claude Light](images/claude-light-preview.png)

### Claude Dark

A rich, warm dark theme that maintains Claude's signature feel.

![Claude Dark](images/claude-dark-preview.png)

## Colors

The theme uses Claude's distinctive color palette:

| Role | Light | Dark |
|------|-------|------|
| Background | `#FAF9F5` | `#141413` |
| Foreground | `#1F1E1D` | `#E8E6DC` |
| Accent (terracotta) | `#D97757` | `#D97757` |
| Keywords | `#D73A83` | `#E87DA8` |
| Functions | `#1F6FE4` | `#85B5E0` |
| Strings | `#26831A` | `#8FA870` |
| Types | `#8A46CE` | `#A872D9` |
| Constants | `#2D8F8F` | `#45B0B0` |
| Parameters | `#B56613` | `#D49A3C` |

## Installation

1. Open **Extensions** in VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for **Claude Inspired Theme**
3. Click **Install**
4. Open **Command Palette** (`Ctrl+Shift+P` / `Cmd+Shift+P`)
5. Select **Preferences: Color Theme**
6. Choose **Claude Light** or **Claude Dark**
